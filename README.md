# zu-vue-cli [![Latest Stable Version](https://img.shields.io/npm/v/zu-vue-cli.svg)](https://www.npmjs.com/package/@trinasolar/zu-vue-cli)

A simple CLI for creating your project

## Install

```shell
$ npm install -g zu-vue-cli
```

## Usage

```shell
# 查看模板列表
$ zu-vue list
# 初始化项目工程
$ zu-vue init <模板名> <项目名>
# cd 项目名
$ cd <项目名>
```
